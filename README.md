# Supernatural Creatures NFT Collection

1,006 NFTs with MP4 animations and WebP fallbacks.

## Structure
- `/mp4/` - Video files (1-1006.mp4)
- `/webp/` - Image fallbacks  
- `/metadata/` - Token metadata JSON

## Usage
Raw URLs:
```
https://raw.githubusercontent.com/YOUR_USER/supernatural-creatures/main/mp4/1.mp4
https://raw.githubusercontent.com/YOUR_USER/supernatural-creatures/main/webp/1.webp
```
